Jual Tiket Masuk Bandara - Aplikasi Desktop Sederhana

Aplikasi ini adalah solusi desktop berbasis Windows Forms untuk mengelola data penjualan tiket masuk bandara. Dibuat dengan C# dan SQL Server, program ini memungkinkan pengguna untuk:

Tambah Data Tiket:
Input nomor tiket, tanggal keberangkatan, asal, dan tujuan penerbangan.
Simpan data ke database SQL Server dengan mudah.
Tampilkan Data Tiket:
Menampilkan semua data tiket dalam format tabel (DataGridView).
Memudahkan pengguna untuk melihat dan menganalisis informasi tiket.
Update Data Tiket:
Edit informasi tiket yang sudah ada, seperti tanggal, asal, atau tujuan.
Update data di database dengan cepat.
Hapus Data Tiket:
Hapus data tiket berdasarkan nomor tiket.
Menjaga database tetap bersih dan terorganisir.
Koneksi Database:
Terhubung ke database SQL Server "BandaraDB" yang terletak di server "DESKTOP-HAIN24I\TIKET".
Menggunakan integrated security.
Antarmuka Pengguna (UI):
Antarmuka yang sederhana dan intuitif dengan TextBox, DateTimePicker, dan DataGridView.
Pesan konfirmasi (MessageBox) untuk setiap operasi (simpan, update, hapus).
Fitur Utama:
CRUD (Create, Read, Update, Delete) operasi data tiket.
Tampilan data tabel yang mudah dibaca.
Koneksi database SQL Server yang stabil.
DateTimePicker untuk memudahkan input tanggal.

Singkatnya: Aplikasi ini memberikan solusi dasar untuk mengelola penjualan tiket bandara, dengan fungsionalitas CRUD dan tampilan data yang sederhana.
